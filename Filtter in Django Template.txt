## أمثلة توضيحية لبعض الفلاتر المفيدة:

* **`title`** — يحوّل كل كلمة إلى حالة "أولى كبيرة"

  ```django
  {{ "hello world"|title }} <!-- → "Hello World" -->
  ```

* **`default`** — يستخدم قيمة افتراضية إذا كان المتغير فارغًا

  ```django
  {{ value|default:"غير محدد" }}
  ```

* **`length`** — يعطي عدد عناصر القائمة أو طول النص

  ```django
  {{ my_list|length }}
  ```

* **`join`** — يجمع عناصر القائمة مع نص محدد

  ```django
  {{ my_list|join:", " }}
  ```

* **`truncatewords` / `truncatechars`** — يختصر النص بعد عدد كلمات أو أحرف معينة

  ```django
  {{ long_text|truncatewords:5 }}
  ```

* **`date`** — ينسق التاريخ بالشكل المناسب

  ```django
  {{ publication_date|date:"D d M Y" }}
  ```

* **`timesince` / `timeuntil`** — يعرض العمر أو المدة حتى تاريخ معين بشكل بشري

  ```django
  {{ some_date|timesince }}
  ```

* **`escape` / `safe`** — للتعامل مع HTML وحماية من XSS

  ```django
  {{ content|escape }}  <!-- يحول النص للعرض الآمن -->
  {{ content|safe }}   <!-- يعرض HTML دون هروب -->
  ```

## الخلاصة

* **Django يوفر مجموعة واسعة من الفلاتر الجاهزة** لتسهيل معالجة المتغيرات ضمن القوالب، مثل التعامل مع النصوص والتواريخ والقوائم والمنطق.
* **يمكن دمج الفلاتر (Chaining)** بسهولة:

  ```django
  {{ text|escape|linebreaks }}
  ```
* **إذا احتجت وظيفة غير موجودة**، يمكنك دائمًا إنشاء فلتر **مخصص** عبر Python داخل مجلد `templatetags` في التطبيق الخاص بك ([Django Project][2], [Real Python][3]).
